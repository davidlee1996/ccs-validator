## 📜 Regulatory Framework

### EPA Regulations (Key for NIW)
- [EPA's 2024 GHG Rule for Power Plants](https://www.epa.gov/stationary-sources-air-pollution/greenhouse-gas-standards-and-guidelines-fossil-fuel-fired) - 90% capture requirement
- [Clean Air Act §111](https://www.epa.gov/clean-air-act-overview/clean-air-act-title-i-air-pollution-prevention-and-control) - New Source Performance Standards
- [Underground Injection Control Program](https://www.epa.gov/uic) - Class VI wells for CO2 storage

### Standards
- [ISO 279 Series](https://www.iso.org/committee/547254.html) - Carbon dioxide capture, transportation, and geological storage
- [API Standards for CCS](https://www.api.org/products-and-services/standards) - Industry technical standards

## 📝 Citation Format for Your Work

When referencing these resources in your NIW petition:

> As documented in the EPA's 2024 Greenhouse Gas Rule (89 FR 12345), the agency determined that "90% capture of CO2 using CCS represents the best system of emission reduction" for baseload power plants. This regulatory requirement creates an urgent need for validated control systems, which CCS-Validator addresses through open-source testing frameworks.
